# 腾讯云翻译功能配置指南

**适用场景：** 本文档适用于使用 Pelr 或其他兼容桌面工具配置腾讯云机器翻译服务的用户。整个配置流程约需 10 分钟。

---

## 一、准备工作：获取 API 密钥

使用翻译功能前，需从腾讯云获取一对 API 凭证——**SecretId** 和 **SecretKey**。

### 1.1 注册腾讯云账号

1. 打开 [腾讯云官网](https://cloud.tencent.com/)
2. 点击右上角 **"免费注册"**
3. 使用微信或 QQ 扫码完成注册

### 1.2 完成实名认证

1. 点击右上角头像 → **账号信息** → [实名认证](https://console.cloud.tencent.com/developer/auth)
2. 按页面指引填写身份信息（支持个人或企业认证）
3. 认证通常在几分钟内审核通过

### 1.3 开通机器翻译服务

1. 登录 [腾讯云控制台](https://console.cloud.tencent.com/)
2. 访问 [机器翻译控制台](https://console.cloud.tencent.com/tmt)
3. 勾选服务协议，点击 **"开通服务"**

> **免费额度说明：** 腾讯翻译 API 每月提供 **500 万字符** 的免费额度，个人日常使用完全够用，无需充值。

### 1.4 创建密钥

需创建一个专门用于翻译的子账号并获取其密钥。

> **注意：** 密钥中的 `SecretKey` **仅在创建时显示一次**，关闭弹窗后无法再次查看。请务必在显示后立即复制并保存至安全位置（如密码管理器或加密文本文件）。

**操作步骤：**

**第一步：创建子用户**

1. 打开 [访问管理 - 用户列表](https://console.cloud.tencent.com/cam)
2. 点击 **"新建用户"** → **"自定义创建"**
3. 类型选择 **"可访问资源并接收消息"**，点击下一步
4. **用户名：** 填写易记的名称，例如 `pelr-translate`
5. **勾选 "编程访问"**，点击下一步

**第二步：授权翻译权限**

1. 在策略搜索框中输入 `TMT`
2. 勾选 **`QcloudTMTFullAccess`**（机器翻译全读写权限）
3. 点击下一步，再点击完成

**第三步：生成密钥**

1. 在用户列表中，点击刚创建的 `pelr-translate` 用户名
2. 切换到 **"API 密钥"** 标签页
3. 点击 **"新建密钥"**，系统将发送验证码至您的手机
4. 输入验证码后，页面将显示：

```
SecretId: AKIDxxxxxxxxxxxxxxxxxxxxxxxxxx
SecretKey: xxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

5. **立即** 将这两个值复制并保存至安全位置。

## 二、在软件中配置翻译功能

获取密钥后，打开桌面软件的设置面板，找到翻译相关配置区域。

### 2.1 填写配置信息

| 配置项         | 填写内容                                                   |
| :------------- | :--------------------------------------------------------- |
| 翻译引擎       | 选择 **腾讯云**                                            |
| SecretId       | 粘贴已保存的 SecretId（以 AKID 开头）                      |
| SecretKey      | 粘贴已保存的 SecretKey                                     |
| 服务地域       | 保持默认 `ap-guangzhou`（广州）即可                        |
| 目标语言       | 翻译后的目标语言，如 `en`（英语）、`ja`（日语）            |
| 源语言         | 可留空，系统将自动检测；也可手动指定                        |

### 2.2 保存确认清单

- [ ] 翻译引擎已切换为腾讯云
- [ ] SecretId 已完整粘贴，末尾无多余空格
- [ ] SecretKey 已完整粘贴，末尾无多余空格
- [ ] 目标语言已正确填写（如 `en`、`ja`、`ko`）

## 四、常见问题

| 问题现象                   | 可能原因               | 解决方法                                                                                   |
| :------------------------- | :--------------------- | :----------------------------------------------------------------------------------------- |
| 提示 "SecretId 未找到"     | SecretId 填写错误或已禁用 | 前往 [API 密钥管理](https://console.cloud.tencent.com/cam/capi) 检查密钥状态         |
| 翻译结果为空或无响应       | 网络连接问题           | 检查本机网络连接；如在公司网络环境下，可尝试切换至移动热点测试                             |
| 翻译内容明显错误           | 源语言设置不正确       | 将源语言设为自动检测（auto）                                                               |
| 不确定密钥是否正确         | 需单独验证             | 使用 [API Explorer](https://console.cloud.tencent.com/api/explorer?Product=tmt&Version=2018-03-21&Action=TextTranslate) 在线调试 |

如问题仍未解决，请查阅腾讯云机器翻译错误码说明文档。

---

## 五、安全提示

- **请勿将 SecretKey 分享给他人**，否则他人可盗用您的翻译额度。
- **请勿将包含密钥的配置文件公开发布** 至网盘、论坛等公开平台。
- 如怀疑密钥已泄露，请 **立即** 前往 [API 密钥管理](https://console.cloud.tencent.com/cam/capi) 禁用旧密钥并创建新密钥。
